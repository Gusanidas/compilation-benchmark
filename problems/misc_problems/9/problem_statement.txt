Create a program to manage users and their cars with a many-to-many relationship. Implement the following features:
- Add a user. Arguments are the user's name and ID. Return true if the user was added successfully.
- Add a car. Arguments are the car's ID, make, and model. Return true if the car was added successfully.
- Assign a car to a user. Arguments are the user's ID and the car's ID. Return true if the assignment was successful.
- Get all cars for a user. Argument is the user ID. Return a comma-separated list of car IDs assigned to the user, in the order they were assigned, without spaces in between.
